Este repositorio contiene un cuaderno digital sobre Ondas Estacionarias en una Cuerda Tensa, elaborado como material académico para el análisis y comprensión del comportamiento de las ondas mecánicas en sistemas vibratorios.

El cuaderno desarrolla conceptos fundamentales relacionados con la formación de ondas estacionarias, modos normales de vibración, frecuencia fundamental y armónicos en cuerdas sometidas a tensión.
